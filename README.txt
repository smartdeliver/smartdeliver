
# smartdeliver - בוט שליחויות

## מה יש כאן?
- `main.py` - קוד הבוט
- `requirements.txt` - ספריות שצריך להתקין
- `.env` - קובץ הגדרות עם טוקן

## איך מעלים ל-Railway?
1. כנס לאתר railway.app ופתח פרויקט חדש.
2. העלה את הקבצים מהתיקיה הזאת.
3. בתפריט Environment הוסף משתנה:
   - KEY: BOT_TOKEN
   - VALUE: הטוקן של הבוט שלך
4. לחץ Deploy וזהו!

הבוט שלך יהיה חי ומוכן לשימוש!
